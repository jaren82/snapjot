# 크롬 웹스토어 등록 — 한국어

## 이름
SnapJot(스냅잣) — 캡처, 주석, 복사

## 짧은 설명 (132자 이내)
영역 찰칵 → 박스·화살표·메모 → 바로 클립보드. 계정도 클라우드도 없이, 스크린샷은 내 컴퓨터를 떠나지 않습니다.

## 상세 설명

화면의 "여기요!"를 한 호흡에 전달하세요.

SnapJot(스냅잣)은 업무 피드백을 위한 일부러 작게 만든 캡처 도구입니다.

⚡ 한 흐름으로 끝
아이콘 클릭(또는 Cmd/Ctrl+Shift+S) → 드래그로 영역 선택(Enter는 전체 화면) → 주석 → 복사. 슬랙·노션·메일에 Cmd/Ctrl+V로 바로 붙여넣기. 편집기 창도, 추가 단계도 없습니다.

🧰 필요한 도구만
• 박스 — 말하려는 영역 표시
• 화살표 스티커 — 하자점검 스티커처럼 정확한 지점을 콕
• 텍스트 라벨 — 깔끔한 라운드 메모칩
• 5가지 기본색 + RGB 자유 선택(스포이드 포함)
• 실행 취소, PNG 저장

🔒 설계부터 프라이빗
• 계정 없음, 가입 없음
• 클라우드 없음 — 캡처는 어디에도 업로드되지 않고 클립보드로만 갑니다
• 최소 권한 — 버튼을 누른 그 순간, 그 탭만 볼 수 있습니다
• 분석·추적·데이터 수집 없음 (개인정보처리방침 참고)

캡처가 공개 URL로 올라가는 도구에 데인 적 있다면, 이 도구가 답입니다.

📝 정직한 안내
• 무료 버전은 캡처 우측 하단에 작은 "created by 스냅잣" 크레딧이 들어갑니다.
• 보이는 화면 영역을 캡처합니다 (스크롤 전체 캡처는 지향점이 아닙니다 — 빠르고 단순한 게 목표).
